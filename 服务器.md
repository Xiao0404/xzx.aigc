# 全栈项目部署
    - 先将后端文件传到服务器
    - 然后再配置数据库
        - 宝塔数据库名默认是小写，上传的文件中的连接数据库代码要修改  前端发送请求的代码也要修改请求地址
    - 然后再配置前端
        前端文件打包到dist 在vite.config.js 中配置  base:'/dist/'
        然后再 npm run build 打包到dist
        然后再上传到服务器
        然后再配置nginx
